# Resources

## General Guides
- [gm0](https://gm0.org/en/latest/index.html) - The most comprehensive guide that covers everything from building techniques to tournament advice.
- [FTC Discord] (https://discord.com/invite/first-tech-challenge) - The most active discussion-based FTC community server, great for up-to-date information and quick answers.
- [Game and Season Materials](https://www.firstinspires.org/resource-library/ftc/game-and-season-info) - The official source for the current season's info, including the Game Manuals which detail game rules.
- [FTC Tutorials](http://ftctutorials.com/en/) - An unofficial guide that delves into robot design, competition strategies, and team management.

## Programming 
- [Learn Java for FTC](https://raw.githubusercontent.com/alan412/LearnJavaForFTC/master/LearnJavaForFTC.pdf) -  A FTC programming guide targeted towards beginners with no preexisting Java knowledge.
- [CTRL ALT FTC](https://www.ctrlaltftc.com/) -  A comprehensive guide to control theory created by FTC #19376 Thermal Equilibrium.
- [Controls Engineering in the FIRST Robotics Competition] - A book that introduces students to the advanced control theory.
- [FTC Dashboard](https://github.com/acmerobotics/ftc-dashboard) - A real-time debugging tool that uses websockets and React. Provides live telemetry graphing and real-time configuration tuning.
- [Easy Open CV](https://github.com/OpenFTC/EasyOpenCV) - Simplifies the use of OpenCV for FTC robots.
- [Road Runner](https://github.com/acmerobotics/road-runner) - A motion planning library ideal for precise autonomous robot movement, permitting intricate path-following while controlling speed and acceleration.
- [Learn Road Runner](https://learnroadrunner.com/) - A step-by-step guide for setting up Road Runner
- [MeepMeep](https://learnroadrunner.com/tools.html#meepmeep) - A sophisticated visualizer for Road Runner paths, perfect for pseudocode drafting and prototyping.
- [RRPathGen](https://github.com/Jarhead20/RRPathGen) - An user-friendly drag-and-drop tool for prototyping Road Runner paths.
