# HardwareMap

| Method                        | Description                                                                                                                                                     |
| ----------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| get(Class, String deviceName) | Retrieves the device with the indicated name which is also an instance of the indicated class or interface. If no such device is found, an exception is thrown. |
| getAll(Class)                 | Returns all the devices which are instances of the indicated class or interface.                                                                                |
